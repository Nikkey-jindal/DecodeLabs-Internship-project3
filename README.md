# ✈ SkyFly – Flight Booking Website

SkyFly is a responsive flight booking website developed using **HTML, CSS, and JavaScript**. It allows users to search for flights, enter passenger details, view a booking summary, and confirm their booking through an interactive interface.

---

## 🚀 Live Demo
https://nikkey-jindal.github.io/DecodeLabs-Internship-project3/

## 📂 GitHub Repository

https://github.com/Nikkey-jindal/DecodeLabs-Internship-project3

## ✨ Features

- Modern and responsive UI
- Flight search by departure and destination
- Displays available flights
- Passenger details form
- Gender selection
- Booking summary
- Fare calculation
- Booking confirmation page
- Random Booking ID generation
- Interactive user experience
- Mobile, tablet, and desktop responsive
